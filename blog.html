=> Discuss the scope of var, let, and const .

Ans: The keywords var, let, and const are used to declare variables in JavaScript. The scope of a variable declared with var is the function scope or the global scope, depending on where the variable is declared. The scope of a variable declared with let is the block scope. The scope of a variable declared with const is also the block scope. There are three different scopes in JavaScript

Global scope: If declared outside of any function, var variables become global and are accessible throughout the entire script.

Function scope: The function scope is the scope of a function. Variables declared in a function are only accessible from within that function.

Block scope: Block scope: Variables declared using the let or const keyword have block scope. They can only be accessed from within the block that they are declared in.

=> What do you mean by REST API? .
  • A REST API (Representational State Transfer Application Programming Interface) is a set of rules and conventions for building and interacting with web services. It is based on the principles of using standard HTTP methods (GET, POST, PUT, DELETE) to perform operations on resources identified by URLs. RESTful APIs are designed to be simple, scalable, and stateless, making them suitable for the web.
  • Example: Think of a REST API as a menu in a restaurant. The menu lists the available dishes you can order, along with a description of each dish. You can use the menu to place an order, specifying what you want to eat (GET request). The restaurant's kitchen prepares the requested dish, and you receive the food you ordered (HTTP response). If you're not satisfied with your order or want to change it, you can ask the waiter to make modifications (PUT or DELETE request). The menu serves as a standardized way for customers to interact with the restaurant's kitchen, just like a REST API provides a standardized way for clients (e.g., web or mobile applications) to interact with a server and its resources over the internet.
